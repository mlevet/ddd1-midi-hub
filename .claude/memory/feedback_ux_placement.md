When controls belong to the same workflow (e.g., Load Bank + Genre filter + Scenes list), group them in the same panel rather than separating them by mode. User called this "elegant."

**Why:** Scenes panel is where the user goes to load patterns — Load Bank and Genre filter belong there, not hidden inside the PatternBank pad mode header.

**How to apply:** When adding a new control, ask "where does the user go when they need this?" and place it there, even if another location seems architecturally cleaner.
